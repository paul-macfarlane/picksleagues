## Notable / Gotchas

- **Driver selection at runtime**: `lib/db/index.ts` uses `@neondatabase/serverless` when `process.env.VERCEL` is set, and `pg` otherwise. Local dev points at docker Postgres (`localhost:5436`), Vercel points at Neon.
- **Migration history is per-environment**: the local docker DB has stale `__drizzle_migrations` from legacy attempts (see PL-004 notes). Neon starts clean, so the first `pnpm db:migrate` against Neon will cleanly apply `0000_cheerful_wolf_cub.sql`.
